# COMPLIANT.EU

COMPLIANT.EU is a food information compliance tool designed to ensure that food products meet the necessary regulatory standards across Europe.

## Features
- Compliance checking
- User-friendly interface
- Comprehensive database of regulations

## Installation
1. Clone the repository
2. Follow the provided instructions for setup and configuration.

## Usage
- Simply input the required food product information to receive compliance feedback.
